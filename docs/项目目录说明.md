# 科吉 Agent 目录说明

## 根目录（常用）

| 路径 | 说明 |
|------|------|
| `main.py` | 启动 FastAPI 服务 |
| `desktop.py` | 桌面壳（webview） |
| `cli.py` | 命令行入口 |
| `config.yaml` | 本地配置（已 gitignore，勿提交） |
| `config.example.yaml` | 配置模板 |
| `setup_deploy.bat` / `一键部署.bat` | 新机安装 venv 与依赖 |
| `launch_keji.bat` / `启动科吉.bat` | 后台启动并打开浏览器 |
| `run_server.bat` / `运行服务.bat` | 带控制台启动（排错） |
| `offline_packages/` | 离线 pip wheel（Git LFS） |
| `scripts/deploy.ps1` | 部署脚本（由 bat 调用） |
| `build.bat` | PyInstaller 打包（产物在 `dist/`，可删后重建） |

## 核心代码

| 路径 | 说明 |
|------|------|
| `core/` | 路由、数据库、RAG、技能注册、安全鉴权与审计 |
| `nanobot/` | Agent 运行时（基于 [HKUDS/nanobot](https://github.com/HKUDS/nanobot)，见 `nanobot/LICENSE`） |
| `web/` | 前端静态资源（`index.html` + `js/` + `css/`） |
| `prompts/` | 系统提示词模板 |

## 数据与运行时（gitignore，保留即可）

| 路径 | 说明 |
|------|------|
| `data/` | SQLite、向量索引、上传文件等 |
| `sessions/` | 对话会话 JSONL |
| `logs/` | 运行日志、审计 `audit.jsonl` |
| `memory/` | Agent 长期记忆 `MEMORY.md`（可空目录） |
| `knowledge/` | 知识库源文件 |
| `venv/` | Python 虚拟环境 |

## 技能与扩展

| 路径 | 说明 |
|------|------|
| `skills/` | 应用内技能库（`core/skills` 扫描） |
| `.claude/skills/` | Cursor/Claude 本地技能（与上表独立） |
| `mcp/` | MCP 服务（如 quack-mcp） |

## 开发脚本

| 路径 | 说明 |
|------|------|
| `scripts/dev/` | 飞书/工具等诊断脚本，非生产入口 |

## 已清理的杂物（勿再放根目录）

- 调试输出：`response*.txt`、`*.log`、`tavily_test.txt`
- 临时测试：`test_*.json`、`test.docx`
- 误建目录：`%SystemDrive%`、`-p`
- 重复前端：`web/_all.js`（已拆分为 `web/js/*.js`）
- 一次性脚本：`reconciliation.py`（对账示例，与科吉无关）
