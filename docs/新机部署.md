# 新机部署（详细版）

适合：**第一次**在一台 Windows 电脑上安装科吉，或从 [GitHub](https://github.com/Zuofeng198/keji) 克隆后的完整流程。

## 一、安装前置软件

1. **Git** + **Git LFS**（克隆含离线依赖包时必需）  
2. **Python 3.12**（64 位，勾选 Add to PATH）— 与仓库 `offline_packages/BUILD_INFO.txt` 一致  
3. **Node.js 18+**（可选，MCP 完整功能）

## 二、获取项目

```powershell
git clone https://github.com/Zuofeng198/keji.git
cd keji
git lfs install
git lfs pull
```

若无 Git、仅有 zip 拷贝：确保文件夹内包含 `offline_packages/pip_wheels`（约 400MB）。

## 三、运行安装脚本

双击 **`setup_deploy.bat`**。

成功标志：提示 Deploy finished，且出现 `venv` 文件夹。

## 四、配置

```powershell
# 若脚本未自动生成，可手动复制：
copy config.example.yaml config.yaml
copy .env.example .env
```

编辑 **`.env`**（必填示例）：

```env
DEEPSEEK_API_KEY=sk-xxxx
KEJI_ADMIN_PASSWORD=你的强密码
```

可选：`TAVILY_API_KEY`（联网搜索）、`OPENAI_API_KEY`（若改用 OpenAI 模型）。

## 五、启动与验证

1. 双击 **`launch_keji.bat`**
2. 浏览器打开 http://127.0.0.1:8000/
3. 用 `admin` + 上面密码登录
4. 发一条测试消息，确认有回复

## 六、局域网给其他电脑用

- 服务已监听 `0.0.0.0:8000`
- 其他电脑浏览器访问：`http://服务器内网IP:8000/`
- 若不通：Windows 防火墙 → 入站规则 → 允许 TCP 8000

## 七、关闭服务

- `run_server.bat` 窗口：`Ctrl+C`
- `launch_keji.bat`：任务管理器结束 `pythonw.exe`
